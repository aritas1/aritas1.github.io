Lightroom face tagging plugin

Demo

Install the plugin

please download (latest version from git) the plugin file into the Adobe Lightroom folder (e.g. C:\Program Files\Adobe\Adobe Photoshop Lightroom 5.7) so that it can run properly.

Lightroom

start Lightroom, and select File -> Plugin Manager. in the lower left corner, click Add, and choose the correct directory (in this case it should be C:\Program Files\Adobe\Adobe Photoshop Lightroom 5.7) Once installed correctly, you'll see Lightroom Facetags setup as a new plugin within Plugin Manager.

Usage

Workflow

  1. Open Lightroom, go to File -> Plugin Extras and click on Lr Facetags.
  2. Wait until detection is done. (May take some time, a progressbar will show up)
  3. Remove boxes that do not mark faces or you do not need (e.g. if you do not know the person), or add boxes whenever a face was not detected.
  4. Add some persons to the training set (see next paragraph).
  5. Run face recognition. (May take some time)
  6. View results. Change names of persons that were not recognized correctly.

Adding images to the training set

There are three possibilities to add a face to the training set: 1. Put a box around a face, click on the name of that person and click "add face". 2. Click on an existing red box. Click on the name of the person and click "change face". 3. If a person was not recognized correctly: Click on the (green) box. Change the name of the person and click "change face".

Red and green boxes

Red boxes mark faces that do not have a name assigned yet.

Green boxes mark faces that already have a name assigned. After running face recognition all the boxes will be green.

Shortcut keys

  • Previous/next image: Arrow left/right
  • Remove face: Delete
  • Add/change face: Insert
  • Clear selection: Escape
